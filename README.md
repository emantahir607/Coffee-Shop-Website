# ☕ Bean Haven Coffee Shop

## 📖 Description

Bean Haven Coffee Shop is a modern and responsive coffee shop website developed using HTML and CSS. It showcases the coffee shop's menu, gallery, customer reviews, and contact information with an attractive user interface.

## ✨ Features

- Responsive Navigation Bar
- Hero Banner
- About Section
- Coffee Menu
- Image Gallery
- Customer Testimonials
- Contact Form
- Footer
- Responsive Design using CSS Media Queries

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

```
Coffee-Shop-Website/
│
├── index.html
├── style.css
├── images/
└── README.md
```

## ▶️ How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open **index.html** in any modern web browser.
